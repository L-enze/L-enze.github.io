project darkspele

Me 2e project die ik heb gemaakt is project darkspele het is een website waar je 3 spellen in kan spelen https://github.com/ROCMondriaanTIN/sd23-p02-project-build-a-game-pablo.git.

dit is een spel in een spel je hebt de keuze uit 3 spellen: 1 brood, kaas en ham 2 zeeslag 3 whack a yous!

brood,kaas en ham: je bent hier om de beurt het doel is om 3 op een rij te krijgen horizontaal verticaal of schuin

per beurt mag je maar 1 vierkant kiezen als je 3 op een rij hebt heb je gewonnen

zeeslag: ieder heeft een kant met schepen je moet elkaars schepen blind proberen te raden waar ze zijn via vierkanten

als je 1 goed heb mag je nog een keer dit gaat door totdat 1 al zijn schepen kwijt is en dus heeft verloren

wack a yous!: je moet hier yous proberen te raken die op je scherm naar voren komt per keer dat je hem raakt krijg je een puntz

als je een aantal punten hebt gehaald binnen een bepaalde tijd word het spel groter waardoor het moeilijker word om yous te raken

de technieken die ik vooral heb gebruikt is event.target.classList zodat je css op kon halen

om bijvoorbeeld als je het klikt de vierkant van kleur kan laten verranderen

talen: html,css en javascript
